# Node API REST
this is an simple example to client report.

# Required Software
* Nodejs
* Express
* Mongodb
* Mongoose
* Babel (ES6+)

# Endpoints
* POST api/creacliente, add new Client.
* GET api/kpideclientes, Show Average age and the standard deviation.
* GET api/listclientes, get all Clients.

# Run the project
In development
```shell
npm run dev
```
